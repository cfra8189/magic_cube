# Magic Cube

![Magic Cube demo](gif/magic-cube.gif.gif)

## Description
A retro-styled Magic Cube fortune-telling web app with interactive 3D cube animations and themed answers. Built with vanilla HTML, CSS, and JavaScript, featuring 1980s Macintosh System 1 aesthetics, glassmorphism effects, and animated GIF backgrounds with light/dark theme support.

## Technologies Used
- **HTML5** - Semantic structure and accessibility
- **CSS3** - Retro-inspired styling with glassmorphism, CSS variables, and animations
- **JavaScript (ES6+)** - Interactive cube functionality and theme management
- **CSS Variables** - Dynamic theming system
- **LocalStorage** - Theme preference persistence
- **Google Fonts** - ChicagoFLF custom font integration

## Features
- **Interactive Magic Cube** - Click the cube or button to reveal fortune answers
- **3D Animations** - Cube rotation and shake effects during fortune telling
- **12 Fortune Answers** - Classic Magic 8-Ball responses with retro styling
- **Light/Dark Theme Toggle** - Seamless theme switching with persistence
- **Glassmorphism Design** - Modern frosted glass effects with backdrop blur
- **Animated Background** - Retro computer GIF with theme-aware filtering
- **Responsive Layout** - Optimized for desktop and mobile devices
- **Custom ChicagoFLF Font** - Authentic 1980s Macintosh typography
- **Keyboard Support** - Enter key submission and navigation
- **Retro Error Alerts** - Themed system messages matching the aesthetic

## Future Features

## Usage
1. Enter your name (optional) in the input field
2. Type your question in the question field
3. Click the Magic Cube or press Enter to get your fortune
4. Toggle between light and dark themes using the switch
5. Enjoy the retro 1980s aesthetic with modern glassmorphism effects
